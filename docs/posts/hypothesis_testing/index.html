Data is the evidence. But is it in your favour?¶

Data, as is, cannot be useful unless we are asking questions which can be partly or fully be answered based on it. The kind of question one puts here can be very broad and narrow at the same time.

What is a hypothesis?¶

A hypothesis is a statement about the population which one believes is true based on 1) hunch 2) scientific theory 3) empirical studies.
